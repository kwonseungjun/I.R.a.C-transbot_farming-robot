# I.R.a.C-transbot_farming-robot
This projet need to using transbot from YAHBOOM robotics and also need open_manipulator produced in ROBOTIS.

This system is ROS1 PROJECT!

Use intel-realsence depth camera for YOLO_V5 dataset.

Entire sytem was controlled in Jetson Nano Dev-board.

This project researched in 2025 [summer], and published on papers at same year.

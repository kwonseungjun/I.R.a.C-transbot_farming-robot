This project presents a ROS1-based autonomous mobile robot platform designed for agricultural environments.
The system integrates vision-based perception, autonomous navigation, and mobile robotic operation using the Transbot platform and Jetson Nano embedded system.
The project was developed during Summer 2025 and later extended toward autonomous mobile manipulation research.

————————————————————————————————————————————————————————————————————————

This project is based on the Transbot platform developed by YAHBOOM Robotics.

This project is based on the ROS1 framework.

An Intel RealSense depth camera was used for YOLOv5-based perception and dataset generation.

The entire system was operated on a Jetson Nano Developer Kit.

The research results were published in a conference paper in 2025.


<'Overview'>

A mobile robot platform aimed at autonomous movement and object collection in an agricultural environment


<'Features'>

- caterpillar drive mobile platform

- Vision-based object perception

- Sensor fusion

- Autonomous outdoor navigation

- Remote control mode


<'Hardware'>

- MCU: Jetson nano dev kit 4GB RAM

- Motor: 520 DC Gear Motor with Encoder 205RPM 333RPM 550RPM

- Motor driver: Transbot expansion board & U2D2

- Camera: intel-realsense & pi-cam

- Battery: 12.6V 4400mAH Lithium-ion (Li-ion) battery


<Software>

- ROS

- Python

- C++

- OpenCV 4.1.1

- YOLO V5.0

- TensorRT
